ShopWave — Project Guide

Autonomous Support Resolution Agent — hackathon-style submission for fictitious retailer ShopWave: 20 mock tickets, ingest → triage → tool-based resolution or escalation → full audit trail.

1. Purpose

StageImplementation
IngestLoads JSON + KB from agentic_ai_hackthon_2026_sample_data-main/.
TriageCategory, urgency, resolvability — rules + optional LLM (Ollama/OpenAI), merged conservatively.
ResolveMock CRM tools: lookups, eligibility, refund, KB, reply.
Escalateescalate(ticket_id, summary, priority) with structured summary.
AuditThought + tool + attempt + result + status → logs/audit_log.json + optional WebSocket UI.

Concurrency: All 20 tickets processed in parallel via asyncio.gather.

2. Hackathon criteria mapping

RequirementImplementation
Multi-step tool chainSession → lookups → policy/KB → reply or escalate (KB padding before escalate when needed).
Fault tolerancewith_retry; optional simulated timeouts/malformed responses; schema validation.
Parallel ticketsagent/runner.py uses asyncio.gather.
ExplainabilityAudit steps with thought; optional llm_triage JSON in logs.

4. Tools (mocks)

Read: get_customer, get_order, get_product, search_knowledge_base

Write: check_refund_eligibility, issue_refund(order_id, amount), send_reply, escalate(..., priority)

Optional audit step: llm_triage when LLM classification is enabled.

5. Optional LLM triage

Classifies category/urgency/resolvability only; refunds and CRM writes stay deterministic. Enable with SHOPWAVE_USE_LLM_TRIAGE=1 plus OLLAMA_BASE_URL or OPENAI_API_KEY. Copy .env.example to .env.

6. Command Center UI

Build with npm install && npm run build. Run uvicorn api.server:app. Open root URL → Run all tickets; WebSocket streams tool_step events.

7. Environment variables

VariableRole
AGENT_SIMULATE_FAULTSSimulate flaky tools (timeouts, bad KB payload)
SHOPWAVE_USE_LLM_TRIAGEEnable LLM triage
OLLAMA_BASE_URL / OLLAMA_MODELLocal Llama via Ollama
OPENAI_API_KEY / OPENAI_MODEL / OPENAI_BASE_URLCloud / compatible API

8. How to run

# CLI
pip install -r requirements.txt
python -m agent.main

# Dashboard
npm install && npm run build
python -m uvicorn api.server:app --host 127.0.0.1 --port 8000
